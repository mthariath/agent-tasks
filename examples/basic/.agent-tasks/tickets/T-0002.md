## Context

The standard needs a usable local command surface to prove that the file model is practical.

## Outcome

The reference tool can initialize a project, validate files, list items, inspect details, and mutate status.

## Acceptance Criteria

- `init`, `validate`, `list`, `show`, and `set-status` exist
- all mutations go through the shared core engine
- command output stays readable in plain terminals
