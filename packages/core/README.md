# @agenttasks/core

Core parser, validator, planner, and mutation engine for `agent-tasks`.

## Install

```bash
npm install @agenttasks/core@next
```

## What It Exposes

- project indexing and validation
- ticket and epic creation/editing helpers
- dependency-aware planning helpers
- execution lifecycle helpers
- YAML/frontmatter parsing utilities
